i-Flix

Built With TMDB API / JSON / JavaScript / Semantic HTML5 / Responsive CSS3

Browse your favorite Movies, TV shows, and actors. Search for specific movies, shows, or actors by using a simple UI. i-Flix is a movie app that allows users to search for movies by title, rating, and release date using the MovieDatabase API.
